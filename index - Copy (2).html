About
This is a prototype for a platform-independent mobile phone application to provide functionality around eRx Express. Transactions with web services are currently only simulated.

The application was written using HTML5 which can be embedded in packages for all mobile phone platforms (Windows, Android, Apple, Blackberry etc.) using Adobe PhoneGap.
